¿Quién soy?

Soy una persona muy responsable, en gran parte porque desde pequeño me enseñaron a cumplir con mis deberes y organizar bien mi tiempo. Gracias a eso, he aprendido a ser disciplinado, a terminar lo que empiezo y a enfrentar mis responsabilidades con compromiso y dedicación.

Soy una persona que le gusta la programacion, desarrollar soluciones eficientes y aprender constantemente nuevas tecnologías. Disfruto el proceso de resolver problemas, optimizar código y crear aplicaciones que sean útiles y funcionales, siempre buscando mejorar la experiencia del usuario y la calidad del software.

- Javier Esneider Pinto Rodriguez

Mis proyectos

Proyecto 1

En este proyecto se observa una hoja de vida, la cual ha sido diseñada para presentar de manera clara y organizada la información personal, académica y profesional, demostrando mi capacidad para estructurar contenido de forma adecuada y aplicar principios de diseño que facilitan la lectura y comprensión de la información.

Proyecto 2

En este proyecto realicé el diseño de un ticket digital para un concierto de Ed Maverick, en el cual integré información relevante del evento como fecha, lugar y acceso. Con este trabajo demuestro mi capacidad para crear diseños atractivos y funcionales, aplicando herramientas de desarrollo web y cuidando los detalles visuales y mejor experiencia.

Proyecto 3

En este proyecto desarrollé una página web para la empresa de turismo “Aventura Total”, en la cual diseñé una interfaz atractiva y funcional que permite mostrar destinos, planes turísticos y servicios ofrecidos, demostrando mi capacidad para crear soluciones digitales adaptadas a las necesidades de una empresa.

Proyecto 4

En este proyecto desarrollé una plataforma web para la gestión de reservas en línea, donde los usuarios pueden registrarse, consultar disponibilidad y realizar reservas de manera rápida y sencilla, demostrando mi capacidad para crear sistemas funcionales aplicando conocimientos en desarrollo web, manejo de bases de datos y diseño de interface.
